# Astraia-Net

1.文件介绍

(1) 服务器 包含大厅服务器源码

(2) 数据库 包含数据服务器源码

(3) 源程序 可以直接运行的控制台程序

2.使用方法:

Astraia.Lobby.exe 大厅服务器

Astraia.Myqsl.exe 数据服务器

3.主要功能:

(1) 大厅房间信息

(2) 玩家加入房间

(3) 游戏房主踢人

4.注意:

(1) 不支持断线重连

(2) 不需要导入Unity

(3) 运行数据服务器需要安装 Mysql

5.框架地址:

[https://github.com/1176892094/JFramework](https://github.com/1176892094/JFramework)
